Progressive Enhansment

Progressive enhansment is when webdevelopers focus on optomizing for content. This is opposed to graceful degridation where a webpage is optomized for the best browsers. The way progressive enhansment is acommpoished is by alowing the html to be usable on its own without css or javascipt. This alows browsers who are not powerful enough to handle the css and javasript to still have acess to the content. Progressive enhansment also allows for html and css to be usable for browsers that cant handle javascrip, but can handle css. Finaly it alowes for browsers who can handle all three to use them to full effect.
